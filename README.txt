Manual de instalación:
Es posible descargar los archivos o clonar el repositorio en github: https://github.com/LorraineRojas/FinalProjectLP.git
Al abrir el IntelliJ IDEA se debe seleccionar el SDK en File -> ProjectStructure
También seleccionar el módulo Trainer en las opciones de compilación Run -> Edit Configurations

Manual de uso:
En la carpeta input, en el archivo entrada.txt se debe pegar el código a analizar en el lenguaje Python.
Después se le oprime al botón Run'Trainer' para correr el programa.
Se debe indicar el nombre de usuario y dar click en el botón "Analizar"

Nota:
El porcentaje de similitud y con quien se tiene la similitud en el código aún no aparece en la interfaz, se puede observar en la consola de IntelliJ IDEA
